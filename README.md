#📅 Calendar App (React + TypeScript)<br><br>

A modern and interactive Calendar Application built using React, TypeScript, and Vite. This app allows users to manage events, view dates, and interact with a clean UI.<br><br>

#🚀 Features<br>
📆 Interactive calendar UI<br>
➕ Add / manage events<br>
⚡ Fast performance using Vite<br>
🎯 Built with TypeScript for better code quality<br>
📱 Responsive design<br>
<br>
📦 Installation & Setup<br><br>

#Follow these steps to run the project locally:<br>

1️⃣ Clone the repository<br>
git clone https://github.com/gaurav7k7/Calendar-app-main.git<br>
cd calendar-app-main<br>

2️⃣ Install dependencies <br>
npm install <br>

<br>
3️⃣ Run the development server <br>
npm run dev<br>
4️⃣ Open in browser<br>
http://localhost:5173<br>
